UBER PLONE

Plone ist ein Web-Content-Management-System, geschrieben in der Programmiersprache Python. Version 1 erschien 2003. Es kann fr Intranet- und Extranet-Anwendungen, als Document Publishing System und als TZT WERDEN. ZA

Es baut auf dem Open Source Application Server Zope und dem zugehrigen Content-Management-Framework auf, das von Tausenden von Entwicklern rund um die Welt untersttzt wird. Plone gilt als das erfolgreichste Projekt, das auf Zope basiert. Mit Plone lassen sich durcfach (u. a. UML-basierend mittels ArchGenXML) eigene Web-Applikationen entwickeln.

INSTALLIERE PLONE AUF DOCKER

Docker nos permite instalar plone de manera muy sencilla con una sola linea de codigo, todo lo demas requiere un poco de conocimientos utilizando CMS , en un principio se hizo un tanto dificil instalar plone en docker debido a la falta de conocimientos utilizando contenedores, pero una vez instalado el uso del mismo es bastante sencillo

docker run -v /home/webplone/:/var/www/htdocs/ -p 8580:8080 --name plonewebpage -d plone

Con esta sencilla linea de codigo se instala la imagen de plone sobre docker, se crea el contenedor del mismo y se configura el puerto siendo en esta ocasion el puerto 8580 ya que el puerto 8080 se encuentra ocupado.
